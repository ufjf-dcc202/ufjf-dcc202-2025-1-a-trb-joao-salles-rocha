# ICEDew Valley

## Projeto
* Criação de um jogo baseado na mecânica de plantação do Stardew Valley

# Principais pontos
* Grade 12x12 para o canteiro
* Interações com o canteiro
* Crescimento variado das sementes
* Coletar plantas prontas
* Comprar novas sementes
* Passagem de tempo
